## 1. Цель Проверки

Провести всестороннюю проверку научного кода проекта **cc_toy_lab** — численного стенда для игрушечных моделей сфер S², S³, S⁶ и их произведения S³ × S⁶ в контексте теории ковариантной компактификации Тома Лоуренса.

### 1.1 Критерии Оценки

Проект будет оцениваться по следующим критериям:

1. **Корректность математических формул** — соответствие реализованных формул аналитическим выражениям из теории
2. **Численная точность** — соответствие численных результатов аналитическим предсказаниям в пределах заявленной точности
3. **Тестовое покрытие** — полнота проверки критических компонентов
4. **Воспроизводимость** — возможность повторить результаты с фиксированными seed
5. **Документация** — полнота и точность документации
6. **Качество кода** — соответствие best practices (type hints, docstrings, структура)
7. **Соответствие целям** — достижение заявленных целей проекта

## 2. Области Проверки

### 2.1 Геометрия (cc_toy_lab/geometry/)

**Файлы для проверки**:
- `analytic_spectra.py` — аналитические формулы спектров Лапласа-Бельтрами
- `sampling.py` — равномерная выборка на сферах (метод Мюллера)
- `graph_laplacian.py` — графовый лапласиан через kNN

**Критические проверки**:
1. ✓ Формула собственных значений: λ_ℓ(S^n_R) = ℓ(ℓ + n - 1) / R²
2. ✓ Формула кратностей для S², S³, S⁶
3. ✓ Сумма Минковского для произведения многообразий: λ(M₁ × M₂) = λ(M₁) + λ(M₂)
4. ✓ Скалярная кривизна: Scal(S^n_R) = n(n-1) / R²
5. ⚠ Численная сходимость графового лапласиана к аналитическому спектру
6. ⚠ Зависимость точности от параметров k (число соседей) и ε (ширина ядра)

**Ожидаемая точность** (из технического отчета):
- S² через FD: 0.1–1% для ℓ ≤ 6
- S² через HEALPix: машинная точность
- S³ через граф-лапласиан: 1–5% для первых 4–10 уровней при N ≥ 5000

### 2.2 Спектральный Анализ (cc_toy_lab/spectral/)

**Файлы для проверки**:
- `anderson.py` — модель локализации Андерсона
- `toy_dirac.py` — toy-версия оператора Дирака
- `metrics.py` — IPR, r-статистика
- `localization.py` — метрики локализации

**Критические проверки**:
1. ✓ IPR (Inverse Participation Ratio): IPR = Σ|ψᵢ|⁴
2. ✓ r-статистика (Oganesyan-Huse): r̃ₙ = min(sₙ, sₙ₊₁) / max(sₙ, sₙ₊₁)
3. ✓ Переход Пуассон → GOE при увеличении беспорядка
4. ⚠ Воспроизводимость с фиксированным seed
5. ⚠ Численная устойчивость eigsh с sigma=0

**Ожидаемые значения**:
- ⟨r̃⟩ Пуассон: 0.386
- ⟨r̃⟩ GOE: 0.531
- IPR делокализованной моды: ≈ 1/N
- IPR локализованной моды: → 1

### 2.3 Физика Радиона (cc_toy_lab/radion/)

**Файлы для проверки**:
- `potentials.py` — потенциалы радиона (A, B, C, D)
- `dynamics.py` — динамическая эволюция
- `mfg.py` — Mean Field Game стабилизация
- `phase_transition.py` — фазовые переходы

**Критические проверки**:
1. ✓ Потенциал A: V_A = a/R²
2. ✓ Потенциал B: V_B = a/R² + bR²
3. ✓ Аналитический минимум для B: R_min = (a/b)^(1/4)
4. ✓ KK-башня: Σ exp(-k/cutoff) √((k/R)² + m₀²)
5. ⚠ Численная производная vs аналитическая
6. ⚠ Кривизна в минимуме (стабильность)
7. ⚠ Сходимость MFG к целевому радиусу

**Физическая осмысленность**:
- Минимум должен быть устойчивым (кривизна > 0)
- KK-башня должна давать положительный вклад
- MFG должна сходиться к 1/R за время O(1/α)

### 2.4 Топология (cc_toy_lab/topology/)

**Файлы для проверки**:
- `betti.py` — числа Бетти
- `persistent.py` — персистентная гомология через GUDHI
- `optional_gudhi.py` — опциональная зависимость

**Критические проверки**:
1. ✓ Числа Бетти для S²: b₀=1, b₁=0, b₂=1, χ=2
2. ✓ Числа Бетти для S³: b₀=1, b₃=1, b₁=b₂=0, χ=0
3. ✓ Числа Бетти для S⁶: b₀=1, b₆=1, остальные 0, χ=2
4. ✓ Формула Кюннета для произведения: b_k(M×N) = Σ b_i(M) b_j(N)
5. ⚠ Устойчивость персистентной гомологии при деформациях

**Ограничения** (из отчета):
- S⁶ через Rips практически невозможна (комбинаторно тяжело)
- Требуется witness-комплекс или альфа-комплекс

### 2.5 Обнаружение Аномалий (cc_toy_lab/discovery/)

## 4. Численная Точность

### 4.1 Сравнение с Аналитикой

**Тесты точности** (запустить вручную):

```python
# Geometry: S² спектр
from cc_toy_lab.geometry.analytic_spectra import sphere_laplacian_eigenvalue
import numpy as np

for ell in range(10):
    analytic = sphere_laplacian_eigenvalue(dimension=2, ell=ell, radius=1.0)
    expected = ell * (ell + 1)
    assert np.isclose(analytic, expected, rtol=1e-10)
    print(f"ℓ={ell}: λ={analytic:.6f} (expected {expected})")
```

```python
# Radion: минимум потенциала B
from cc_toy_lab.radion.potentials import find_minimum, analytic_radius_b, potential_b, DEFAULT_RADION_PARAMS

min_numeric = find_minimum(potential_b, DEFAULT_RADION_PARAMS)
min_analytic = analytic_radius_b(DEFAULT_RADION_PARAMS)
error = abs(min_numeric.radius - min_analytic) / min_analytic
print(f"Numeric: R={min_numeric.radius:.10f}")
print(f"Analytic: R={min_analytic:.10f}")
print(f"Relative error: {error:.2e}")
assert error < 1e-6, f"Numerical minimum error {error:.2e} exceeds tolerance"
```

### 4.2 Скейлинг по R

**Проверка λ ∝ 1/R²**:
```python
radii = [0.5, 1.0, 2.0]
for R in radii:
    lambda_R = sphere_laplacian_eigenvalue(dimension=3, ell=1, radius=R)
    scaled = lambda_R * R**2
    print(f"R={R}: λ={lambda_R:.6f}, λ·R²={scaled:.6f}")
    assert np.isclose(scaled, 3.0, rtol=0.01)  # ℓ=1 для S³: 1*(1+2)=3
```
