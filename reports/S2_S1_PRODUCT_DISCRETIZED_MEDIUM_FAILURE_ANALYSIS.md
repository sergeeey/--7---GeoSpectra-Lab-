## Итоговая классификация (для medium-ветки, подвыборка W>0)

1. **Доминирующий сигнал:** **transition-regime sensitivity при `W=4`** (все 6 кейсов; при `W=8/12` для `ring` — 0 non-robust).  
2. **Вторичный сигнал:** **localized ring / α=0 product-discretized caveat** для **4 / 6** строк.  
3. **Дополнительно:** два **`alpha=0.5`**, **`s1_size=8`**, симметричные по спектру кейсы (`q=±1`) с полным **`fail`** — укладываются в **слабый disorder + малая размерность + ring**, без распространения на большие `W` в этом прогоне.

**Не выводим:** континуумную компактификацию, S6/S3×S6, SM, физическую хиральность, обход Witten/Lichnerowicz.

Исторические ограничения legacy v2/v3 **не пересматриваются** и не «стираются» этим отчётом.
