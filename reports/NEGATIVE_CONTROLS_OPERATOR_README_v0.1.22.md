# Negative Controls — Инструкция для Оператора

**Для кого:** Человек который будет запускать вычисления на удалённом сервере или другом компьютере  
**Зачем:** Мой компьютер перегревается и выключается во время расчётов. Нужно выполнить оставшиеся 4 части на более мощной машине.

---

## Что нужно сделать (кратко)

1. Скачать проект с GitHub
2. Установить Python и библиотеки
3. Запустить 4 команды (batches 3, 4, 5, 6)
4. Отправить результаты обратно

**Время:** ~4 часа работы компьютера (можно оставить и уйти)

---

## Требования к компьютеру/серверу

**Минимум:**
- Linux (Ubuntu, Debian) или Windows с Python
- 8 ядер процессора, лучше 16
- 32 GB оперативной памяти, лучше 64 GB
- 20–50 GB свободного места на диске
- Интернет (для скачивания проекта)

**Видеокарта НЕ нужна.**

**Подходящие варианты:**
- Облачный сервер (AWS, Google Cloud, Azure) — ~$10 за весь прогон
- Университетский кластер (если есть доступ) — бесплатно
- Компьютер родственника/друга с хорошим охлаждением
- Google Colab Pro ($10/месяц) — самый простой вариант

## Пошаговая инструкция (Linux)

### Шаг 1: Скачать проект

Открыть терминал и выполнить:

```bash
git clone https://github.com/sergeeey/N-7-GeoSpectra-Lab.git
cd N-7-GeoSpectra-Lab
git checkout main
git pull origin main
```

**Проверка:** Команда `git log --oneline -3` должна показать последний коммит 32d8298.

### Шаг 2: Установить Python и библиотеки

```bash

source .venv/bin/activate
pip install --upgrade pip

```

**Проверка:** Команда `python -c "import numpy, scipy; print('OK')"` должна вывести `OK`.
